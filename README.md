# CTBC_RWD_Hongkong
RWD 基礎實踐：香港旅遊一頁式網站

專案說明 (Project Description)
此專案旨在利用純 HTML 與 CSS 實作一個單頁式 (One-Page) 的響應式網頁。內容以香港旅遊作為主題，專注於基礎的網頁結構、視覺呈現以及跨裝置的介面適應性。排版上輔以 Bootstrap 框架，確保介面元件的標準化與開發效率。

核心技術與設計理念
1. 技術棧 (Tech Stack)
● 語言： HTML5, CSS3
● 佈局： Bootstrap 5 (排版輔助), RWD (Responsive Web Design) - 透過 Media Queries 實現響應式佈局。

2. 設計理念 (Design Philosophy)
● 介面適應性： 確保網站在桌面、平板和手機等三種主流裝置尺寸下，內容佈局能夠自動且無損地重新排列。
● 視覺平衡： 運用圖片與文字的比例，在單頁內呈現主題氛圍，避免在行動裝置上出現水平滾動條。
● 代碼基礎： 程式碼結構清晰，為後續更複雜的專案奠定基礎。
